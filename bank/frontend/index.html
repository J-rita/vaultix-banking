Security

Your security is our foundation

Security is built into every part of Vaultix, from the login page to the database.

Password Hashing

Passwords are secured using bcrypt. Your plain-text password is never stored anywhere in the system.

ACID Transactions

All transfers are wrapped in a single database transaction. If any step fails, everything is reversed and no money is moved.

Audit Logging

Every login, transfer, and account change is recorded automatically in the database so there is always a clear record of what happened.
